# VidSave Pro - Backend Server

This repository contains the backend Python server for VidSave Pro, which handles video extracting, downloading, and converting using `yt-dlp` and `ffmpeg`.

## One-Click Deployment

Click the button below to deploy this backend directly to your Render.com account for free:

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/gtest7206-dot/vidsave-pro)

Once deployed, copy the backend URL Render gives you and paste it into the `API_BASE` configuration in `js/downloader.js` on your frontend.
